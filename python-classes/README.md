# Python Classes Project

This project covers Object-Oriented Programming (OOP) in Python.

## Concepts Learned
- Classes and Objects
- Attributes (public, private)
- Methods
- Property getters and setters
- Data validation
- Encapsulation

## Tasks
0. Empty class Square  
1. Square with size  
2. Size validation  
3. Area of a square  
4. Access and update private attribute  
5. Printing a square  
6. Coordinates of a square
