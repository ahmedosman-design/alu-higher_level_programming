# Python - If/Else, Loops, Functions

This project is part of the ALU Higher Level Programming curriculum.

## Concepts Covered

- if / elif / else statements
- for loops
- range()
- ASCII manipulation with ord() and chr()
- nested loops
- Python functions
- returning values

## Files

0-positive_or_negative.py — prints whether a random number is positive, negative, or zero.

1-last_digit.py — prints the last digit of a random number with conditions.

2-print_alphabet.py — prints the lowercase alphabet.

3-print_alphabt.py — prints alphabet except q and e.

4-print_hexa.py — prints numbers 0–98 in decimal and hexadecimal.

5-print_comb2.py — prints numbers from 00 to 99.

6-print_comb3.py — prints all combinations of two digits.

7-islower.py — checks if a character is lowercase.

8-uppercase.py — converts a string to uppercase.

9-print_last_digit.py — prints the last digit of a number.

10-add.py — adds two integers.

11-pow.py — calculates power of a number.

12-fizzbuzz.py — prints FizzBuzz from 1 to 100.
