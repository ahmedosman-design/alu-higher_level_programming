# Python - Hello, World

This directory contains projects introducing the Python programming language.

## Tasks

**Task 0:** 0-run - Shell script that runs a Python script. The Python file name is stored in the environment variable $PYFILE.

**Task 1:** 1-run_inline - Shell script that runs Python code stored in the environment variable $PYCODE.

**Task 2:** 2-print.py - Python script that prints "Programming is like building a multilingual puzzle".

**Task 3:** 3-print_number.py - Python script that prints the integer stored in the variable number followed by "Battery street".

**Task 4:** 4-print_float.py - Python script that prints a float with 2 decimal places.

**Task 5:** 5-print_string.py - Python script that prints a string 3 times followed by its first 9 characters.

**Task 6:** 6-concat.py - Python script that prints "Welcome to Holberton School!".

**Task 7:** 7-edges.py - Python script that extracts parts of a string.

**Task 8:** 8-concat_edges.py - Python script that prints "object-oriented programming with Python".

**Task 9:** 9-easter_egg.py - Python script that prints "The Zen of Python".
