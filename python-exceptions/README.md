# Python Exceptions

This project focuses on handling errors in Python using try and except.

## Concepts Covered
- Exception handling (try / except / finally)
- Safe execution of code
- Error messages
- Raising exceptions
- Working with lists and division safely

## Tasks
0. Safe list printing  
1. Safe integer printing  
2. Print and count integers  
3. Safe division  
4. Divide lists  
5. Raise exception  
6. Raise exception with message  
7. Safe integer print with error  
8. Safe function execution  
9. Magic calculation
